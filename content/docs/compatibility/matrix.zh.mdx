## 已知问题

### SGLang Reasoning Token

SGLang 对 DeepSeek-R1 等推理模型的 `reasoning_tokens` 返回格式与 OpenAI 标准不同，可能导致统计数据缺失。

**临时解决方案**：在 NexusGate 配置中启用 `normalize_reasoning_tokens` 选项。

### MindIE Usage 统计

MindIE 部分版本的 `usage` 字段返回不完整。

**建议**：升级到最新版本 MindIE 或在 NexusGate 中启用 Token 计数功能。
