欢迎使用 NexusGate！NexusGate 是一个开源的 AI 网关服务，旨在为您提供统一的 AI 模型调用管理能力。

## 什么是 NexusGate？

NexusGate 作为中间层，连接您的应用与各种 AI 模型服务商，提供：

- **统一接口**：兼容 OpenAI API 格式，一套接口对接所有模型
- **用量统计**：追踪 Token 消耗、请求次数等关键指标
- **多模型支持**：支持 OpenAI、Anthropic、DeepSeek、阿里云百炼等服务商
- **下游兼容**：适配各类 AI 工具和应用

## 核心特性

### OpenAI 兼容 API

NexusGate 完全兼容 OpenAI API 格式：

- 所有支持 OpenAI API 的工具都可以直接使用
- 无需修改现有代码，只需更改 API 地址
- 支持 Chat Completions、Models 等核心接口

### 多模型路由

支持配置多个上游模型服务商：

- **OpenAI**：GPT-4o、GPT-4o-mini 等
- **Anthropic**：Claude 4 Opus/Sonnet
- **DeepSeek**：DeepSeek-V3、DeepSeek-R1
- **阿里云百炼**：Qwen 系列
- **本地部署**：vLLM、SGLang、TGI 等

### 用量追踪

精确统计每次请求的资源消耗：

```json
{
  "usage": {
    "prompt_tokens": 100,
    "completion_tokens": 50,
    "total_tokens": 150,
    "reasoning_tokens": 30
  }
}
```
