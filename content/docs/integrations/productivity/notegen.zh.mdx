NoteGen 是一款 AI 驱动的笔记生成工具，可以自动将各种内容转换为结构化的笔记。

## 工具简介

- **GitHub**：[NoteGen](https://github.com/nicepkg/notegen)
- **支持平台**：Web、桌面应用
- **特点**：
  - 内容自动整理
  - 多格式支持
  - AI 总结提取
  - 知识图谱

## 配置步骤

### 1. 安装 NoteGen

#### 桌面应用

从 GitHub Releases 下载对应平台的安装包。

#### Web 版本

访问 NoteGen 在线版本或自托管部署。

### 2. 打开设置

1. 启动 NoteGen
2. 点击设置图标
3. 选择「AI 配置」

### 3. 配置 OpenAI 兼容 API

| 配置项 | 值 |
|--------|-----|
| API 类型 | OpenAI Compatible |
| API 地址 | <ServerUrl path="/api/v1" /> |
| API Key | <ApiKeyLink /> |
| 模型 | `gpt-4o` |

## 使用示例

### 网页转笔记

1. 复制网页链接
2. 粘贴到 NoteGen
3. AI 自动提取关键内容生成笔记

### 视频转笔记

1. 输入视频链接（YouTube、Bilibili 等）
2. NoteGen 提取字幕
3. AI 生成结构化笔记

### PDF 转笔记

1. 上传 PDF 文件
2. AI 分析文档结构
3. 生成带目录的笔记

### 音频转笔记

1. 上传音频文件或录音
2. 自动转录
3. AI 整理成笔记格式

## 高级配置

### 自定义笔记模板

配置笔记生成模板：

```yaml
模板名称: 技术文章
格式: |
  # {{title}}

  ## 概述
  {{summary}}

  ## 关键点
  {{key_points}}

  ## 技术细节
  {{details}}

  ## 参考链接
  {{references}}
```

### 配置不同场景的模型

```yaml
网页提取: gpt-4o-mini  # 快速
长文档处理: gpt-4o     # 准确
学术论文: claude-sonnet-4-20250514  # 专业
```

### 配置提取规则

```yaml
提取规则:
  - 标题: 自动识别
  - 作者: 提取作者信息
  - 日期: 提取发布日期
  - 标签: AI 自动生成
  - 摘要: 限制 200 字
```

## 常见问题

### Q: 提取内容不完整？

1. 检查源内容是否完整
2. 调整提取参数
3. 尝试使用更强的模型

### Q: 笔记格式不理想？

1. 自定义笔记模板
2. 调整 AI 提示词
3. 手动编辑调整

### Q: 视频转录不准确？

1. 确保视频有字幕
2. 对于无字幕视频，使用语音识别服务
3. 检查音频质量

### Q: 如何批量处理？

1. 使用批量导入功能
2. 或使用 API 进行自动化处理

## 导出格式

NoteGen 支持多种导出格式：

- **Markdown**: 适合 Obsidian、Notion
- **HTML**: 适合网页展示
- **PDF**: 适合分享打印
- **JSON**: 适合程序处理
