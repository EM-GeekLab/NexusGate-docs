Sider 是一款浏览器侧边栏 AI 助手，可以在浏览任何网页时快速调用 AI 进行问答、翻译、总结等操作。

## 工具简介

- **官方网站**：[sider.ai](https://sider.ai/)
- **支持平台**：Chrome、Edge、Firefox、Safari
- **特点**：
  - 侧边栏快速访问
  - 网页内容理解
  - 多种 AI 功能
  - 快捷键支持

## 配置步骤

### 1. 安装扩展

#### Chrome / Edge

1. 访问 [Chrome 网上应用店](https://chrome.google.com/webstore)
2. 搜索「Sider」
3. 点击安装

#### Firefox

1. 访问 [Firefox 附加组件](https://addons.mozilla.org/)
2. 搜索「Sider」
3. 点击安装

### 2. 打开设置

1. 点击浏览器工具栏的 Sider 图标
2. 登录或创建账户（如需要）
3. 进入设置页面

### 3. 配置自定义 API

1. 在设置中找到「自定义 API」或「OpenAI API」
2. 填入配置：

| 配置项 | 值 |
|--------|-----|
| API 端点 | <ServerUrl path="/api/v1" /> |
| API Key | <ApiKeyLink /> |
| 模型 | `gpt-4o` |

## 使用示例

### 快速问答

1. 按 `Ctrl+P`（或点击图标）打开侧边栏
2. 在输入框输入问题
3. 获取 AI 回答

### 网页总结

1. 打开任意网页
2. 打开 Sider 侧边栏
3. 点击「总结页面」

### 选中文本操作

1. 选中网页上的文本
2. 点击浮动的 Sider 图标
3. 选择操作（翻译、解释、改写等）

### 写作辅助

1. 在任意输入框中写作
2. 选中文本
3. 使用 Sider 润色、扩展或精简

## 高级配置

### 配置多个 AI 服务

Sider 支持配置多个 AI 服务：

```yaml
服务 1: NexusGate GPT-4o
服务 2: NexusGate Claude
服务 3: NexusGate DeepSeek
```

### 自定义 Prompt 模板

创建常用的 Prompt 模板：

```yaml
模板: 代码解释
内容: |
  请解释以下代码的作用：

  ```
  {{selected_text}}
  ```

  请说明：
  1. 代码功能
  2. 关键逻辑
  3. 可能的改进点
```

### 配置快捷键

| 功能 | 默认快捷键 |
|------|------------|
| 打开侧边栏 | `Ctrl+P` |
| 快速提问 | `Ctrl+Shift+P` |
| 翻译选中 | `Ctrl+Shift+T` |

### 配置网站规则

为特定网站配置不同行为：

```yaml
规则:
  - 网站: github.com
    默认操作: 代码解释

  - 网站: arxiv.org
    默认操作: 论文总结

  - 网站: twitter.com
    默认操作: 翻译
```

## 功能列表

### 内容理解

- **总结页面**: 提取网页关键信息
- **解释内容**: 深入解释复杂概念
- **问答**: 基于页面内容回答问题

### 写作辅助

- **改写**: 重新表述内容
- **扩展**: 丰富内容细节
- **精简**: 压缩冗余内容
- **语法检查**: 修正语法错误

### 翻译功能

- **即时翻译**: 选中文本快速翻译
- **全页翻译**: 翻译整个页面
- **多语言**: 支持多种语言对

### 创作功能

- **生成文案**: 根据描述生成内容
- **头脑风暴**: 生成创意想法
- **大纲生成**: 创建内容大纲

## 常见问题

### Q: 侧边栏打不开？

1. 检查扩展是否已启用
2. 尝试刷新页面
3. 检查快捷键是否冲突

### Q: API 调用失败？

1. 验证 API 端点格式
2. 检查 API Key 是否正确
3. 确认 NexusGate 服务状态

### Q: 总结结果不准确？

1. 确保页面内容已完全加载
2. 尝试使用更强的模型
3. 网页内容过长时可能被截断

### Q: 如何同步设置？

使用 Sider 账户登录后，设置会自动同步到其他设备。
