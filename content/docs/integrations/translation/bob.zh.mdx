Bob 是一款 macOS 平台上优秀的翻译和 OCR 软件，支持划词翻译、截图翻译等功能，可以接入多种翻译服务。

## 工具简介

- **官方网站**：[ripperhe.gitee.io/bob](https://ripperhe.gitee.io/bob/)
- **支持平台**：macOS
- **特点**：
  - 划词翻译
  - 截图翻译
  - 输入翻译
  - 支持多种翻译服务

## 配置步骤

### 1. 安装 Bob

#### App Store（推荐）

在 Mac App Store 搜索「Bob」并安装。

#### Homebrew

```bash
brew install --cask bob
```

### 2. 安装 OpenAI 翻译插件

Bob 需要安装插件来支持 OpenAI 兼容 API：

1. 打开 Bob 偏好设置
2. 进入「插件」页面
3. 搜索并安装「OpenAI Translator」插件

### 3. 配置 NexusGate API

1. 进入「服务」页面
2. 点击「+」添加服务
3. 选择「OpenAI Translator」
4. 填入配置：

| 配置项 | 值 |
|--------|-----|
| API URL | <ServerUrl path="/api/v1/chat/completions" /> |
| API Key | <ApiKeyLink /> |
| 模型 | `gpt-4o-mini` |

{/* TODO: 添加 Bob 配置截图 */}
{/* ![Bob 配置](./images/bob-config.png) */}

### 4. 设置为默认翻译服务

1. 在服务列表中将配置好的服务拖到最上方
2. 或在「翻译」设置中选择默认服务

## 使用示例

### 划词翻译

1. 选中要翻译的文本
2. 按下快捷键（默认 `⌥ + D`）
3. 弹出翻译结果

### 截图翻译

1. 按下快捷键（默认 `⌥ + S`）
2. 框选要翻译的区域
3. 自动 OCR 并翻译

### 输入翻译

1. 按下快捷键（默认 `⌥ + A`）
2. 输入要翻译的文本
3. 回车获取翻译结果

## 高级配置

### 自定义 Prompt

在插件设置中可以自定义翻译 Prompt：

```
你是一位专业的翻译专家。请将以下文本翻译成{{to}}，要求：
1. 保持原文的语气和风格
2. 专业术语保持准确
3. 翻译要自然流畅

原文：{{text}}
```

### 配置多个服务

可以配置多个翻译服务用于不同场景：

- **日常翻译**：gpt-4o-mini（快速、经济）
- **专业翻译**：gpt-4o（准确度高）
- **学术翻译**：claude-sonnet-4-20250514（长文本）

### 快捷键设置

在「偏好设置」→「快捷键」中自定义：

| 功能 | 默认快捷键 |
|------|------------|
| 划词翻译 | `⌥ + D` |
| 截图翻译 | `⌥ + S` |
| 输入翻译 | `⌥ + A` |

## 常见问题

### Q: 翻译失败显示网络错误？

1. 检查 API URL 格式是否正确
2. 确认 NexusGate 服务可访问
3. 验证 API Key 是否有效

### Q: 翻译结果不理想？

1. 尝试使用更强大的模型（如 gpt-4o）
2. 调整自定义 Prompt
3. 检查源语言和目标语言设置

### Q: OCR 识别不准确？

Bob 的 OCR 功能与翻译 API 无关，可以：
1. 提高截图清晰度
2. 选择更大的文字区域
3. 调整 OCR 引擎设置

### Q: 如何查看请求日志？

在 Bob 的「偏好设置」→「高级」中开启调试模式。
